# 🀄 麻将游戏 (Mahjong Game)

一个基于 HTML5 的单机麻将游戏，可以与 AI 对战。

## 🎮 在线游玩

访问：https://lizhipeng1.github.io/web-games/mahjong/

## ✨ 功能特点

- 🎯 单机对战 AI
- 🀄 完整麻将牌组（108 张牌）
- 🎨 精美的麻将牌面设计
- 📱 响应式设计，支持手机和电脑
- 🎲 摸牌、出牌完整流程
- 🏆 胜负判定

## 🎲 游戏规则（简化版）

1. 点击"新游戏"开始
2. 玩家先摸一张牌（14 张手牌）
3. 点击选择要出的牌，然后点击"出牌"
4. AI 自动摸牌和出牌
5. 先胡牌者获胜

## 🛠️ 技术栈

- HTML5
- CSS3（渐变、动画）
- JavaScript（原生）

## 📁 文件结构

```
mahjong/
└── index.html    # 游戏主文件
```

## 🚀 本地运行

直接双击 `index.html` 文件即可在浏览器中打开游戏。
